JEliza - Systemvorraussetzungen
Mircosoft Windows Linux
  • Mircosoft Windows 98, ME, 2000, NT, XP oder Windows Vista
  • Beliebige Linux-Distribution ab Kernel 2.4
  • Qt 4.3
Allgemein
  • 40MB freier Festplattenspeicher
  • 200 MB RAM
  • 800Mhz Prozessor (empfohlen, aber auch weniger möglich)
  • Internetanschluss (empfohlen, nicht zwingend)
  • Lautsprecher (empfohlen für Sprachausgabe, nicht zwingend)
